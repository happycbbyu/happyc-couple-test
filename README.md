# 🐰 해피씨쀼 신혼부부 진단 테스트

신혼부부를 위한 사랑·돈·대화 30문항 진단 테스트

## 🌟 진단 항목

- 💕 **사랑의 언어** (10문항) — 게리 채프먼의 5가지 사랑의 언어 기반
- 💰 **돈 성향** (10문항) — 안정/밸런스/공격 5단계 세부 진단
- 💬 **대화 방식** (10문항) — 회피/방어/감정표출/소통 4유형

## ✨ 주요 기능

- 남편/아내 선택 + 닉네임 입력
- 옵션 순서 랜덤 표시 (유형 예측 방지)
- 결과 페이지: 점수 차트, GPT 활용 프롬프트, 응답 요약
- 이미지 저장 + 링크 공유
